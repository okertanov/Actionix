Actionix
========

OS Specific
-----------

    /System/Library/CoreServices/Menu Extras


Plug-in ideas
-------------
 - Applications list
 - Chrome tabs list
 - CPU/Mem indication
 - Battery health
 - Processes list
 - Gmail API tags
 - Gmail API notes
 - Gmail API tasks
 - Clipboard monitor
 - Code snippets
 - Remote Mac Lock/Unlock from iPhone
 - Remote app control
 - Apps scripting
 - Site monitoring for down, new content
